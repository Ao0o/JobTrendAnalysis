# 需求：

爬取各城市某个职位招聘岗位的数量、薪资、技能要求、岗位要求、岗位职责、工作地点分析职业发展趋势

- 从招聘网站爬取就业信息。爬取各城市特定职业招聘的数量、薪资、技能要求、岗位要求、岗位职责、工作地点、福利。并存储于MySQL数据库中。
- 对爬取的数据进行数据分析

## 技术栈

- Scrapy框架爬虫
- MySQL存储
- Redis实现分布式爬虫
